#vant-app

## 简介
vue-cl3构建，基于vue.js和vant 实现。

## 前序准备
本项目技术栈基于 ES2015+、vue、vuex、vue-router 、vue-cli 、axios 和 vant。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build:prod
```

### Lints and fixes files
```
npm run lint
```
